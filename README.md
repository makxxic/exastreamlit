## Project Overview
The Carbon Footprint Calculator allows users to:
- Input daily activities (transportation, energy use, diet)
- Receive AI-powered personalized carbon footprint assessment
- Access actionable tips for reducing emissions
- Earn rewards through gamification and progress tracking
- Track environmental impact over time

The project supports both **individuals and organizations** looking to take measurable action against climate change.

## Features
- Real-time carbon footprint measurement
- AI-driven recommendations tailored to user behavior
- Gamification and reward system
- Simple, intuitive UI/UX for accessibility
- Exportable reports for personal or corporate use
- Educational modules on sustainable living

## Tech Stack
- **Frontend:** React / HTML / CSS / Bootstrap  
- **Backend:** Node.js / Express / Python (for AI modules)  
- **Database:** PostgreSQL / Supabase  
- **AI/ML:** Python, TensorFlow / scikit-learn  
- **Hosting & Deployment:** Vercel / Netlify / Supabase  

## Installation
1. Clone the repository:  
```bash
git clone https://github.com/makxxic/CarbonFootprintCalc.git
```
Navigate to the project directory:

```bash
cd CarbonFootprintCalc
```
Install dependencies:

``` bash
npm install
```
Set up the environment variables for Supabase and OpenAI API keys.

Run the application:

```bash
npm start
```
Usage
Open the app in your browser (usually at http://localhost:3000).

Register or log in.

Fill in your daily activity inputs.

View your carbon footprint dashboard.

Follow the AI recommendations to reduce your carbon impact.

Earn points and badges as you progress.
